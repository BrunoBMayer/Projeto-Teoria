# Análise de Tempo de Execução: QuickSort em Python e C

Este projeto tem como propósito investigar e comparar o desempenho do algoritmo QuickSort implementado em Python e em C, considerando diferentes tamanhos de entrada. Os experimentos e gráficos gerados fazem parte das atividades da disciplina de Teoria da Computação.

## Funcionalidades

- Executa o mergesort 20x em arrays de 1000, 10000 e 50000.
- Cada array é preenchido aleatoriamente, totalmente ordenado e inversamente ordenado.
- Ao final é gerado um CSV para os dados.
- Você pode inserir ambos os CSVs no colab para gerar gráficos.
